Tailscale笔记

记录Tailscale组网的一些

Nov 16, 2023

Nas 端

运行 Tailscale，充当 exit node，配合家庭软路由 openwrt，实现其他组网设备利用全隧道模式扶墙。

创建启动容器：

docker run -d \
--name=tailscaled \
--network=host \
-v /var/lib:/var/lib \
-v /var/run:/var/run \
-v /dev/net/tun:/dev/net/tun \
-e TS_USERSPACE=0 \
-e TS_AUTHKEY=tskey-auth-... \
-e TS_ROUTES=192.168.10.0/24 \
-e TS_STATE_DIR=/var/lib/tailscale \
-e TS_SOCKET=/var/run/tailscale/tailscaled.sock \
-e TS_EXTRA_ARGS="--accept-routes --advertise-exit-node --reset" \
-e TZ=Asia/Shanghai \
--cap-add=NET_ADMIN \
--cap-add=NET_RAW \
tailscale/tailscale

增加 Tailscale 启动配置：

docker exec tailscaled tailscale up --advertise-exit-node --accept-dns=false --advertise-routes=192.168.10.0/24 --accept-routes

Vps 端

因 Tailscale 官方的中转服务期都在国外，所以使用国内 Vps 搭建 Derp 中转服务器，以优化组网件设备互相访问。

创建启动容器同 Nas 端（防止 derp 服务被白嫖）。

创建启动 Derp 容器：

docker run --restart always \
--name derper -p 12345:12345 -p 3478:3478/udp \
-v /var/run/tailscale/tailscaled.sock:/var/run/tailscale/tailscaled.sock \
-e DERP_ADDR=:12345 \
-e DERP_CERTS=/app/certs \
-e DERP_VERIFY_CLIENTS=true \
-d ghcr.io/yangchuansheng/ip_derper

其他终端（iOS、Android…)

应用市场（美区 AppStore, Play Store)下载 Tailscale 客户端，登录账户，选择 Using exit node，开启 VPN 服务。

Android 端可利用 Tasker，连接家庭 Wi-Fi SSID 后自动断开 Tailscale，操作如下：

  • 连接对应 SSID，断开 Tailscale

  • 断开对应 SSID, 连接 Tailscale
